## Short Description (132 chars max)

## Detailed Description
ScholarSync is a scholarly paper reading queue manager for researchers, students, and academics.

Save papers directly from academic websites:
- arxiv.org
- nature.com
- ieee.org
- jstor.org
- sciencedirect.com
- doi.org

Features:
- One-click save from any supported academic site
- Right-click context menu to save DOI links
- Automatic metadata extraction (title, authors, DOI)
- Syncs with your ScholarSync web dashboard and iOS app
- Secure authentication with your ScholarSync account

Free to use with a ScholarSync account. Pro subscribers get unlimited paper storage, PDF uploads, and more.

## Steps to Publish
1. Create a Chrome Developer account at https://chrome.google.com/webstore/devconsole ($5 one-time)
2. Run `bash extension-store/package.sh` to package the extension
3. Add real icon PNGs (16, 32, 48, 128px) to the images/ folder
4. Zip the extension folder
5. Upload at the developer console
6. Fill in the store listing details above
7. Add screenshots
8. Submit for review (typically 1-3 business days)
